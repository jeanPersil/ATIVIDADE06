Atividade 06

Objetivo

Desenvolver um sistema de monitoramento que utilize um microcontrolador ESP32 para detectar o estado de um botão. O projeto deve enviar o status do botão para um banco de dados em tempo real (Firebase Realtime Database) e permitir a visualização desse status a partir de qualquer dispositivo conectado à internet.

Funcionalidades

O sistema deve se conectar a uma rede Wi-Fi para permitir a comunicação com o Firebase. O botão deve ser configurado como uma entrada digital que, quando pressionado, altera seu estado. O estado do botão deve ser enviado para o Firebase como um valor binário (0 ou 1), onde:

0 = representa "não alarmante"

1 = representa "pressionado"
